### 변수의 이름을 지을 때는 다음과 같이 지켜야 할 몇가지 규칙이 있다.

- 변수명은 숫자로 시작할 수 없다.
- `_`(underscore) 와 `$` 문자 이외의 특수문자는 사용할 수 없다.
- 자바의 키워드는 변수명으로 사용할 수 없다. (예: int, class, return 등)

### 자바 변수의 키워드
